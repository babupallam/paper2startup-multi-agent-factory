# Paper2Startup – Multi-Agent Startup Factory

Turn research papers into **startup-ready business blueprints** using specialized AI agents.

Paper2Startup is a **multi-agent AI application built with Google AI Studio**, designed to help researchers, founders, builders, and innovation teams rapidly explore commercialization opportunities from academic research.

The app transforms uploaded research papers into:

- startup opportunities
- market & ICP analysis
- monetization models
- MVP roadmaps
- moat & risk analysis
- action plans
- investor-style feasibility reports
- Markdown + PDF exports

# Features

## Multi-Agent Startup Generation
Specialized AI agents collaborate to convert research ideas into market-ready startup concepts.

## Founder Modes
Choose the commercialization path that best fits your goal:

- Bootstrapped SaaS
- VC Scale Startup
- Enterprise B2B
- Research Licensing
- API Product
- Hackathon MVP

## Deep Opportunity Reports
Each generated opportunity includes:

- Executive Summary
- Market & ICP
- Monetization
- MVP Roadmap
- Pitch Deck
- Moat & Risk
- Action Plan
- Research Evidence
- Feasibility Scoring

## Export System
Export full reports as:

- Markdown
- Optimized PDF

Both exports include **all sections without omitting hidden tabs**.

# Run Locally

## Prerequisites
- Node.js
- npm

## Setup

1. Install dependencies
```bash
npm install
````

2. Create environment file

```bash
.env.local
```

3. Add your Gemini API key

```env
GEMINI_API_KEY=your_api_key_here
```

4. Run the development server

```bash
npm run dev
```
